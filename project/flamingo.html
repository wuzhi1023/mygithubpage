Motion-Controlled Exhibition Display

The project is my work during my intern at FlamingoEDA. It is about the design and implementation of a motion-controlled museum exhibition system.
We Designed the interacting progress, using Leap Motion, NFC and Transparent OLED screen for enriched interaction experience.

I was responsible for the design and implementation of the desktop program, including user interface with OpenFrameworks.
